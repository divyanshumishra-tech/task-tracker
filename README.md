A robust, multi-tenant backend REST API for managing organizational tasks, built with Spring Boot and Spring Security. This application provides a scalable foundation for registering organizations, onboarding admin users, and securely tracking projects.

## ✨ Features
* **Multi-Tenant Architecture:** Associates users and tasks with specific organizational subdomains.
* **Secure Registration:** Handles new company onboarding alongside initial admin user creation.
* **Custom Security Configuration:** Configured Spring Security pipeline for stateless API communication.
* **Data Persistence:** Relational database mapping using Spring Data JPA and Hibernate.

## 🛠️ Tech Stack
* **Language:** Java 17+
* **Framework:** Spring Boot 3.x
* **Security:** Spring Security
* **Database:** MySQL
* **Build Tool:** Maven
